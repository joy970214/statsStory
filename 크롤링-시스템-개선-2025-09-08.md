# 국토교통부 통계포털 크롤링 시스템 대폭 개선 (2025-09-08)

## 문제 상황

### 1. 데이터 검사 버튼 오류
**문제**: "데이터 검사" 버튼 클릭 시 "데이터를 찾을 수 없습니다" 오류 발생
**원인**: 
- 크롤링은 성공했지만 실제 데이터 필드가 모두 빈 객체 `{}`로 저장됨
- IBSheet JavaScript 기반 동적 데이터를 정적 HTML 파싱으로 처리
- 데이터 검색 시 저장된 제목이 너무 일반적("국토교통 통계누리")

### 2. 크롤링 방식의 근본적 문제
**문제**: 요구사항과 실제 구현이 완전히 불일치
**원인**:
- 메타데이터 수집이 "통계정보" 탭만 처리 ("관련용어" 탭 누락)
- `#sFormId` 셀렉트의 통계표명별 개별 수집 미구현
- 조건부 데이터 수집 로직 부재
- 조회 버튼 클릭 없이 데이터 추출 시도

### 3. 다양한 URL 형식 미지원
**문제**: 하나의 URL 패턴으로만 개발되어 범용성 부족
**원인**:
- 날짜 형식: YYYY, YYYY-MM, YYYYMM 혼재
- 통계표 패턴: "_시도별", "_연도별", "시·군·구별" 등 다양
- 각 통계마다 다른 조건부 처리 필요

### 4. 범용 날짜 형식 지원 시스템

```python
async def _detect_date_format(self, driver) -> str:
    """날짜 입력 필드 형식 자동 감지"""
    start_element = driver.find_element(By.ID, "sStart")
    placeholder = start_element.get_attribute("placeholder") or ""
    value = start_element.get_attribute("value") or ""
    
    if "-" in placeholder or "-" in value:
        return "YYYY-MM"
    elif len(placeholder) == 6 or len(value) == 6:
        return "YYYYMM"  
    elif len(placeholder) == 4 or len(value) == 4:
        return "YYYY"

async def _collect_data_with_date_range(self, driver, table_name: str, date_format: str):
    """형식별 5년치 데이터 수집"""
    if date_format == "YYYY":
        start_value = "2020"
        end_value = "2025"
    elif date_format == "YYYY-MM":
        start_value = "2020-1" 
        end_value = "2025-9"
    elif date_format == "YYYYMM":
        start_value = "202001"
        end_value = "202509"
```

```python
async def _click_search_button(self, driver):
    """다양한 조회 버튼 패턴 지원"""
    button_selectors = [

        "//a[contains(@onclick, 'doSearch')]"
    ]
    
    # 실패 시 JavaScript 직접 호출
    driver.execute_script("if (typeof doSearch === 'function') doSearch();")
```

## 실제 분석된 URL 패턴들

### 확인된 최신통계목록 URL들:
1. **공동주택현황** (`hRsId=419&hFormId=5882`): YYYYMM 형식
2. **자동차등록대수** (`hRsId=58&hFormId=5498`): YYYYMM 형식, **"_시도별"** 패턴
3. **주택보급률** (`hRsId=29&hFormId=2100`): **YYYY 형식**, "_시도별", "_연도별" 패턴
4. **미분양현황** (`hRsId=32&hFormId=2082`): YYYYMM 형식, **"시·군·구별"** 패턴
5. **주택건설착공** (`hRsId=471&hFormId=5386`): YYYY-MM 형식, "월계" 패턴
