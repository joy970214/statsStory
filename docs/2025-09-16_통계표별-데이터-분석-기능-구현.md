## 📋 **작업 요청사항**

사용자 요청: "기본통계현황분석에 통계표별분석 버튼을 만들어달라는게 아니고 기본통계현황분석 화면에서 수집된 통계표명 목록이 있는데 이거가 실제 통계표명들로 수집되게하고 밑에 데이터들이 통계표명 별로 볼수 있게 해달라는거야"

---

## 🔍 **발견된 문제들**

### **문제 1: 통계표명이 더미 데이터로 표시**
- **현상**: 기본통계현황분석에서 "수집된 통계표 목록"에 실제 통계표명이 아닌 더미/일반적인 제목만 표시
- **원인**: `processStatisticsData` 함수에서 실제 `table_name` 정보를 제대로 활용하지 못함
- **영향**: 사용자가 실제 수집된 통계표들의 이름을 알 수 없음

### **문제 2: 통계표별 데이터 분리 기능 없음**
- **현상**: 모든 통계표의 데이터가 합쳐져서 표시됨
- **원인**: 통계표별로 데이터를 필터링하는 기능이 구현되지 않음
- **영향**: 특정 통계표의 데이터만 보고 싶어도 전체 데이터만 확인 가능

### **문제 3: 통계표 선택 인터페이스 부재**
- **현상**: 통계표 목록이 단순 텍스트로만 표시되어 상호작용 불가
- **원인**: 클릭 가능한 UI 요소로 구현되지 않음
- **영향**: 사용자가 원하는 통계표를 선택할 수 없음

### **문제 4: TypeScript 타입 안전성 문제**
- **현상**: 통계 계산 과정에서 undefined 객체 접근 시도
- **원인**: 변동계수 계산에서 여러 번 함수 호출하며 null check 부족
- **영향**: 컴파일 오류 발생으로 애플리케이션 빌드 실패

## ✅ **해결방안 및 구현사항**

### **해결방안 1: 실제 통계표명 추출 및 표시**

**구현 내용**:
```typescript
// EnhancedBasicStatisticsViewer.tsx - processRealData 함수 수정
const dataByTable: Record<string, any[]> = {};
if (rawData.raw_data && Array.isArray(rawData.raw_data)) {
  rawData.raw_data.forEach((stat: any, statIndex: number) => {
    const tableName = stat.table_name || `테이블 ${statIndex + 1}`;
    if (!dataByTable[tableName]) {
      dataByTable[tableName] = [];
    }
    dataByTable[tableName].push(stat);
  });
}
setRawDataByTable(dataByTable);
```

**결과**: 
- ✅ 실제 수집된 통계표명들이 정확히 표시
- ✅ `table_name` 필드에서 실제 통계표 이름 추출
- ✅ 통계표별로 데이터 그룹화하여 상태 관리

### **해결방안 3: 통계표별 데이터 필터링 기능**

**구현 내용**:
```typescript
// 선택된 통계표의 데이터만 필터링하는 함수
const getFilteredData = () => {
  if (!selectedTableName || !processedStats) return processedStats?.sample_data || [];
  
  return processedStats.sample_data.filter(item => 
    item.source_table === selectedTableName
  );
};

// 선택된 통계표의 통계 계산 함수
const getSelectedTableStats = () => {
  if (!selectedTableName || !rawDataByTable[selectedTableName]) {
    return processedStats?.numeric_stats;
  }

  const tableData = rawDataByTable[selectedTableName];
  const numericValues: number[] = [];
  
  // 해당 통계표의 숫자 데이터만 추출하여 통계 계산
  tableData.forEach(stat => {
    Object.entries(stat.data || {}).forEach(([key, value]) => {
      // 값 파싱 및 숫자형 데이터 필터링
      const parsedValue = typeof value === 'string' && value.includes("'value'") 
        ? JSON.parse(value.replace(/'/g, '"'))
        : { value, unit: 'text', raw: value };

      if (parsedValue.unit === 'number' && typeof parsedValue.value === 'number') {
        numericValues.push(parsedValue.value);
      }
    });
  });

  // 통계 계산 (평균, 중위수, 표준편차 등)
  return {
    total: numericValues.reduce((sum, val) => sum + val, 0),
    mean: numericValues.reduce((sum, val) => sum + val, 0) / numericValues.length,
    median: numericValues.sort((a, b) => a - b)[Math.floor(numericValues.length / 2)],
    max: Math.max(...numericValues),
    min: Math.min(...numericValues),
    count: numericValues.length,
    std_dev: Math.sqrt(/* 표준편차 계산 */)
  };
};
```

**UI 업데이트**:
```typescript
// 기초통계 지표를 선택된 통계표 데이터로 업데이트

  {formatNumber(getSelectedTableStats()?.mean || 0)}
</div>

// 데이터 샘플 테이블도 필터링된 데이터로 업데이트

  </tr>
))}
```

**결과**:
- ✅ 선택된 통계표의 데이터만으로 기초통계 계산 (평균, 중위수, 최대/최소값 등)
- ✅ 데이터 샘플 테이블에 선택된 통계표의 데이터만 표시
- ✅ 통계표 선택 시 모든 수치가 실시간으로 업데이트
- ✅ 섹션 제목에 선택된 통계표명 표시

### **해결방안 4: TypeScript 타입 안전성 확보**
