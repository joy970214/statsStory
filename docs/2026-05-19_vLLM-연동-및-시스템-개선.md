## 4. Excel + CSV 이중 다운로드 시스템

### 배경
- 크롤링 시 Excel만 수집하면 pandas가 다중 헤더를 `Unnamed: N`으로 파싱하는 문제 발생
- AI 분석 품질 저하 원인

### 해결 방법
- 크롤링 시 **Excel(사용자 다운로드용)** + **CSV(AI 분석용)** 동시 수집
- 파일 다운로드 모달의 라디오 버튼 선택자 활용
  - Excel: `input[name='settingRadio'][value='xlsx']`
  - CSV: `input[name='settingRadio'][value='csv']`

### StatData 모델 변경
```python
downloaded_file_path: Optional[str] = None  # Excel (사용자 다운로드)
csv_file_path: Optional[str] = None          # CSV (AI 분석)
```

### AI 서비스 처리 방식
- `csv_file_path` 존재 시 CSV 원본 텍스트 그대로 AI에 전달
- `_build_csv_raw_prompt()` 전용 프롬프트 추가
- Unnamed 컬럼 문제 자연스럽게 해결 (AI가 원본 구조 직접 파악)
